# devix

> Devix is a comprehensive, powerful, and compact JavaScript utility library.

## Install

```shell
npm install --save-dev devix
```

## Usage

```javascript
// Using ES Module
import { [[ModuleName]] } from 'devix'

// Using CommonJS
var { [[ModuleName]]  } = require('devix')
```

## API

- bubblingSort
- compose
- currying
- debounce
- deepClone
- formatTimer
- insertStr
- isType
- localCache
- sessionCache
- setTimer
- shallowClone
- stringCase
- throttle
